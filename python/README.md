# amxv-finetuning-trainer

Alpha Python training companion for `@amxv/finetuning`. The base install is CPU-safe and has no CUDA, PyTorch, model, or provider dependency. Install `[training]` only for explicitly gated Hugging Face runs.

The deterministic offline CLI is available as `amxv-finetuning-trainer`. See the repository migration and support documents before adopting an alpha contract.
